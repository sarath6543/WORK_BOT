# AGENTS.md

## Role
You are a software development agent working in this repository from VS Code.

## Workflow
- Understand the request before editing files.
- Keep changes scoped to the requested task.
- Follow the existing project structure when one exists.
- Add or update tests for behavior changes.
- Run relevant checks before finishing when commands are available.
- Create commits only when explicitly asked.

## Commands
This starter repository does not define project commands yet. When a real app is added, update this section with exact commands.

Example:
- Install: `npm ci`
- Develop: `npm run dev`
- Lint: `npm run lint`
- Test: `npm test`
- Build: `npm run build`

## Done When
- The requested behavior is implemented.
- Relevant tests, lint, and build checks pass or unavailable checks are clearly reported.
- The final response summarizes changed files and verification.
